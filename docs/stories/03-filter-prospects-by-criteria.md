# User Story: 3 - Filter Prospects by Criteria

**As a** Relationship Manager,
**I want** to filter prospects by city, sector, network, and clusters,
**so that** I can focus on clients within my area of responsibility or interest.

## Acceptance Criteria

* Filter panel is accessible from the main dashboard
* Filters available: City, Sector, Network, Clusters
* Multiple filters can be applied simultaneously
* Prospect list updates immediately when filters are applied
* Active filters are clearly indicated in the UI
* Filters can be easily cleared or reset

## Notes

* Filters should be intuitive and follow Nuvama's branding guidelines
* The main panel displays filters alongside top prospects and suggested actions
