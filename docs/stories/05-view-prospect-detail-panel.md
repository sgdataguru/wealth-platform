# User Story: 5 - View Prospect Detail Panel

**As a** Relationship Manager,
**I want** to view a detail panel showing key metrics, alerts, and relevance explanations for a prospect,
**so that** I can understand the full context before engaging with the client.

## Acceptance Criteria

* Right-side panel displays when a prospect is selected
* Panel shows key metrics relevant to the prospect
* Active alerts are prominently displayed
* Clear explanation of why this prospect is relevant (signal breakdown)
* Panel can be collapsed or dismissed
* Information is presented in a scannable format

## Notes

* The purpose of this data is not to replace relationships but to improve timing and provide better context for engagement
* The design should be simple and intuitive for Relationship Managers
