# User Story: 30 - Variable Capital Company (VCC) Reporting

**As a** Relationship Manager / Fund Administrator,
**I want** to manage and report on holdings within Variable Capital Company (VCC) structures,
**so that** I can support clients using these modern, flexible investment vehicles.

## Acceptance Criteria

* The system supports **VCC** as a specific entity type in the client hierarchy and relationship graph.
* Specific reporting modules designed for VCC sub-funds (Cellular structures).
* Consolidation of wealth across multiple sub-funds within a single VCC umbrella.
